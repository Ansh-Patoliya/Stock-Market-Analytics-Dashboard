# 📈 Stock Market Analytics Dashboard (Power BI)

## 📌 Overview

The **Stock Market Analytics Dashboard** is a comprehensive Power BI project designed to analyze stock market data using technical indicators, trend analysis, momentum indicators, volatility metrics, and market statistics.

The dashboard is built using **5-minute OHLCV (Open, High, Low, Close, Volume)** market data and provides traders, analysts, and investors with interactive insights into price movements and market behavior.

---

# 🎯 Project Objectives

* Analyze stock market price movements.
* Identify bullish and bearish trends.
* Monitor momentum using technical indicators.
* Measure market volatility.
* Analyze candlestick behavior.
* Build an interactive financial analytics dashboard using Power BI.

---

# 📊 Dashboard Pages

## 1. Market Overview

Provides a high-level summary of the market.

### KPIs

* Latest Close
* Daily Return (%)
* Highest High
* Lowest Low
* Total Volume
* Average Volume

### Visualizations

* Price with Moving Averages (MA20, MA50, MA200)
* Trading Volume

---

## 2. Trend Analysis

Analyzes overall market direction using moving averages.

### KPIs

* Current Trend
* Golden Cross Count
* Death Cross Count
* Average MA20 Distance

### Visualizations

* Trend Distribution
* MA20 Distance Over Time
* Golden Cross Events
* Death Cross Events

---

## 3. Momentum Indicators

Measures market momentum.

### Indicators

* RSI (14)
* MACD
* MACD Signal
* MACD Histogram

### Features

* Overbought & Oversold Zones
* MACD Crossover Analysis
* Histogram Trend Analysis

---

## 4. Oscillators & Trend Strength

Analyzes oscillator-based indicators.

### Indicators

* Stochastic Oscillator (%K & %D)
* Williams %R
* Money Flow Index (MFI)

### Features

* Overbought/Oversold Detection
* Oscillator Trend Analysis

---

## 5. Volatility Analysis

Measures market volatility using multiple indicators.

### Indicators

* Average True Range (ATR 14)
* Daily Trading Range
* Rolling Volatility (5, 10, 20)

### Table

* Top 10 Most Volatile Candles

---

## 6. Market Statistics

Provides statistical insights into candlestick behavior.

### KPIs

* Bullish Candle %
* Bearish Candle %
* Average Candle Body
* Average Upper Wick
* Average Lower Wick

### Visualizations

* Average Return by Trading Day
* Average Return by Trading Hour
